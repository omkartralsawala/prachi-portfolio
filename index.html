01 · IOT PLATFORM · 2022 · FULL-STACK

Smart Society

An IoT platform that unifies security, visitor management and resident communication for a residential society.

React Native Django Django REST Firebase FCM MySQL IoT
## the_problem

Housing societies run on a patchwork — WhatsApp groups for notices, paper registers at the gate, phone calls to approve visitors. Nothing is real-time, and nothing leaves an auditable trail.

## the_approach
  • ▹React Native app with role-based views for residents, guards and admins
  • ▹Django + Django REST backend orchestrating visitor approvals, notices and amenity bookings
  • ▹Real-time push notifications for gate approvals and security alerts via Firebase
  • ▹Every gate event written to a centralized, auditable log
## impact
  • ▹Collapsed four disconnected workflows into a single app
  • ▹Visitor approval went from phone-tag to a one-tap push
  • ▹Guards and residents finally share one real-time source of truth

02 · MACHINE LEARNING · 2022 · ML + BACKEND

Brain Tumor Detection

A browser-based tool that screens MRI scans for tumors with a convolutional neural network.

Python TensorFlow / Keras OpenCV Flask NumPy
## the_problem

First-pass MRI screening is manual and radiologist-bound. An automated triage step can flag suspicious scans faster and more consistently — a helpful second pair of eyes.

## the_approach
  • ▹OpenCV preprocessing pipeline: skull-stripping, normalization and augmentation
  • ▹CNN classifier trained in TensorFlow/Keras on labeled MRI datasets
  • ▹Flask web app: upload a scan → preprocess → inference → prediction with a confidence score
## impact
  • ▹End-to-end demo running entirely in the browser, upload to result
  • ▹A reproducible training pipeline for iterating on accuracy

DailyCup
